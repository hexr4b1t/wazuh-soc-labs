# wazuh-soc-labs

This repository documents hands-on SOC labs built using Wazuh SIEM.
The labs focus on detection, visibility, and alerting across Windows and Linux systems.

## Labs
- Wazuh setup and architecture ✅
- File Integrity Monitoring (FIM) ✅
- Network Intrusion Detection using Suricata
- Vulnerability detection
- Malicious command detection
- SSH brute-force detection
- Malware detection using VirusTotal

## Environment
- Wazuh Manager: Ubuntu Server
- Agents: Windows 10 , Linux
- Network: Bridged VirtualBox lab

## Disclaimer
This repository is for learning and documentation purposes only.
